# キンペコ販売促進サイト - 仕様書

## プロジェクト概要
個人でブリードしている熱帯魚（キンペコ）の販売促進を目的としたウェブサイト。

## 技術スタック
- **フレームワーク**: Next.js (App Router)
- **デプロイ**: Vercel
- **バージョン管理**: GitHub
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS

## サイト構成

### ページ構成

#### 1. トップページ（/）
- ヒーローセクション（キンペコの写真とキャッチコピー）
- 最新ブログ記事の紹介（3-5件）
- SNSリンク（YouTube、Twitter/X、Instagramなど）
- 委託販売ショップへのリンク

#### 2. 自己紹介ページ（/about）
- ブリーダーとしての自己紹介
- キンペコへの想い・こだわり
- ブリーディング環境の紹介

#### 3. 血統紹介ページ（/bloodline）
- 血統の詳細説明
- 血統図の表示（可能であれば）
- 血統の特徴やこだわりポイント

#### 4. ブログページ（/blog）
- ブログ一覧ページ
- ブログ詳細ページ（/[blog-slug]）
- カテゴリ・タグ機能（任意）

#### 5. リンク集ページ（/links）
- YouTubeチャンネルへのリンク
- SNSへのリンク（Twitter/X、Instagram、Facebookなど）
- 委託販売ショップへのリンク
- QRコード生成機能（各ページで表示可能）

## 機能要件

### 必須機能
- ✅ ブログ投稿・表示機能（Markdown対応推奨）
- ✅ SNSリンク集
- ✅ レスポンシブデザイン（モバイル対応）
- ✅ ページネーション（ブログ一覧）
- ✅ SEO最適化（メタタグ、OGPタグ）

### オプション機能
- 🔲 ブログの検索機能
- 🔲 ブログのカテゴリ・タグフィルタ
- 🔲 画像ギャラリー
- 🔲 閲覧数の表示
- 🔲 パンくずリスト

## デザイン要件

### カラースキーム
- **メインカラー**: 濃紺（例: #1a237e, #283593, #3949ab）
- **アクセントカラー**: ゴールド（例: #d4af37, #f4d03f, #ffd700）
- **背景色**: 白またはライトグレー
- **テキスト色**: ダークグレーまたは黒

### UI/UX方針
- シンプルで見やすいレイアウト
- かっこいいが過度ではないデザイン
- モダンなデザイントレンドを取り入れる
- アクセシビリティを考慮（コントラスト比、フォントサイズなど）
- スムーズなページ遷移・アニメーション

### レイアウト構成
- **ヘッダー**: ロゴ、ナビゲーションメニュー
- **フッター**: コピーライト、SNSリンク
- **サイドバー**: 不要（シンプルに）

## コンテンツ管理

### ブログ管理方法
- **Markdownファイル**
  - `/content/blog/` ディレクトリにMarkdownファイルを配置
  - フロントマターでメタデータを管理（タイトル、日付、タグなど）

### 画像管理
- `/public/images/` ディレクトリに配置
- または、外部画像ホスティングサービス（Cloudinary、Imgixなど）

## SEO対策
- 適切なメタタグ設定
- OGP（Open Graph Protocol）タグ
- 構造化データ（JSON-LD）
- サイトマップ生成
- robots.txt設定

## パフォーマンス
- 画像最適化（Next.js Imageコンポーネント使用）
- コード分割・最適化
- Lighthouse スコア目標: 90以上
