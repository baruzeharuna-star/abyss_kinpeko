# 動画背景ファイル

このディレクトリに水をイメージした背景動画を配置してください。

## 推奨設定

- **ファイル形式**: MP4 (H.264コーデック推奨)
- **解像度**: 1920x1080 または 4K (お好みに応じて)
- **ファイル名**: `water-background.mp4` (または任意のファイル名)
- **ファイルサイズ**: できるだけ軽量化（5MB以下推奨）

## 使用方法

1. 動画ファイルをこのディレクトリに配置
2. `app/page.tsx`の`VideoBackground`コンポーネントの`src`プロパティを更新
3. テクスチャ背景のコメントアウトを解除

## 無料の動画素材サイト

- **Pexels Videos**: https://www.pexels.com/videos/
- **Pixabay Videos**: https://pixabay.com/videos/
- **Coverr**: https://coverr.co/

「water」「ocean」「aquatic」などのキーワードで検索してください。
